Star WebPRNT provides you a multi platform solution to print directly from various devices including PCs and tablets via a web browser.
Just embed JavaScript in this SDK package into your HTML file, and you can control your printer.
It allows you to print receipts using your POS printer only with an equivalent knowledge of web programming. No printer drivers or plug-ins are required.

This document provides information about the StarWebPRNT.
It provides guidelines for our customers to build the printing application.
