     StarWebPRNT  -  Ver 0.3.0
      Readme ファイル                                    2014/01/15

_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/_/

本ソフトウェアは、PC、タブレット等多様なデバイスからWebブラウザを通して印刷を行う、
マルチプラットフォームを実現します。
プリンターの制御は、このSDKにパッケージされているJavaScriptをHTMLファイルに埋め込むだけで行え、 
Webプログラミングを行うのと同等の知識でPOSプリンターからのレシート印刷が可能となります。
また、プリンターを制御するためのドライバーやプラグインをインストールする必要もありません。


=========
対応機種
=========


    FVP10    (FW Ver1.5以降) (*1)
    TSP650II (FW Ver1.2以降) (*1)
    TSP700II (FW Ver4.1以降) (*1)
    TSP800II (FW Ver1.6以降) (*1)

    ASR10    (Linux OS Ver1.6以降)

    (*1)対応I/FはLAN(HE07X/08X Ver3.1.0以降)になります。
